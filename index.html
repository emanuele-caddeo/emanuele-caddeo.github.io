AI Tool · Work in Progress

LLM Knowledge Editing Chatbot

A chatbot interface that integrates parametric knowledge editing directly into natural language interaction. The tool allows users to select and load any model from the Hugging Face library and apply knowledge edits conversationally within the chat. Currently under development, the system aims to make model editing accessible without requiring direct manipulation of model parameters or external tooling.

Master's Thesis · Paper

Supporting Rollback in Parametric Knowledge Editing

Research project and academic paper investigating rollback mechanisms in parametric knowledge editing for large language models. The work explores how edits applied to LLM parameters can be safely reverted, enabling more reliable knowledge management. This line of research supports the transition from traditional database-based knowledge systems toward knowledge infrastructures built directly on top of large language models.
